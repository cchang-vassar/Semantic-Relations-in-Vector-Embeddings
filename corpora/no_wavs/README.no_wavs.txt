IBM Debater(R): Recorded Debating Dataset (V4) + Annotated General-Purpose Claims
====================================================================
This dataset contains 200 audio speeches and their automatic and manual transcripts, as described in [3].
In addition, it includes an annotation layer of General-Purpose Claims, as described in [1].
====================================================================
V3 of this dataset is described in [2], and includes 400 additional audio speeches.
V2 of this dataset is described in [3], which further details the recording of the set of speeches used here.
V1 of this dataset is described in [4], which further details about the speech recording process.
====================================================================
Folders (each folder contains a detailed description of the data):
	speeches: text files containing the ASR and manual transcripts of the speeches
	GPR-KB-55: Annotations performed on the speeches, using the General-Purpose Rebuttal Knowledge Base (see [1]).
====================================================================
